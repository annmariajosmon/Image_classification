# Celebrity Image Classification using CNN

## Overview

This project is a deep learning-based celebrity image classification system developed using Python, OpenCV, TensorFlow, and Keras. The application classifies celebrity images into multiple categories using Convolutional Neural Networks (CNNs).

The project includes image preprocessing, model training, evaluation, and prediction, making it suitable for computer vision and AI-based image recognition applications.

---

## Features

- Celebrity image classification using CNN
- Deep learning-based image prediction
- Image preprocessing using OpenCV
- Multi-class celebrity recognition
- Model training and evaluation
- Prediction using trained `.h5` model
- Real-time image visualization

Open the notebook:

```bash

```

Run all cells to:
- Load the trained model
- Preprocess images
- Predict celebrity classes
- Display prediction results
